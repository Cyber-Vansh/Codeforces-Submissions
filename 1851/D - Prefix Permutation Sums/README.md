Its array of prefix sums — $$$[1, 6, 8, 12, 15]$$$. prefix sums.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2 \cdot 10^5$$$.</p></div><div class="output-specification"><div
